# Nossa Praia Moments V1

Primeira versão pública do Guestbook Digital da Nossa Praia Eventos.

## O que faz
- Página responsiva para celular
- Apresentação do evento
- Formulário de recado
- Campo para foto, áudio ou vídeo
- Botão para WhatsApp
